## schweets official intventory and customer service platform 

Theme: Inventory + ordering platform
Core entities: Products + Orders

## platform description
 - web-based ordering and management platform where customers place orders online, and staff and admins manage products, inventory, users, and order fulfilment.

## users
- schweets admin
    - login
    - log out 
    - manage product and inventory 
    - manage users 
    - oversee all orders 

- schweets staff
    - login 
    - log out 
    - view customer orders 
    - update status 
    - track order fulfilment 
    - communicate with customers 

- schweets customers
    - sign up (required for payment)
    - customer authentication and accounts
    - places order 
    - make payments
    - view submitted order status 

## problem
- Schweets needs a central system where customers can place orders and internal staff can manage inventory and track order fulfilment efficiently.

## success criteria 
- The system is successful when customers can place orders online, and Schweets staff can manage products, view orders, and update order statuses in one central platform.


## MVP Goal
- Enable online order placement for customers and basic order and inventory management for internal users.

## Application Pages / Screens (MVP)
### Authentication
- Register
- Login

### Customer Pages
- Products Listing
- Create Order
- Order Confirmation
- My Orders (Order Status View)

### Staff Pages
- Orders List
- Order Detail View
- Update Order Status

### Admin Pages
- Product Management (Create / Update / Delete)
- Inventory Management (Update Stock)
- User Management (Create / Remove Staff & Admin Users)


## MVP Features (By Role)
### Customer
- View products
- Create order
- Add products with quantities
- Submit order
- View order status using order reference

### Staff
- View orders
- View order details
- Update order status

### Admin
- Login
- Create, update, and delete products
- Update stock quantities
- Create and remove internal users (admin/staff)

## Out of Scope (Post-MVP)
- Online payments
- Automated notifications (email, WhatsApp, SMS)
- Real-time order tracking (GPS/shipping integration)
- Blogs / CMS features
- Analytics and reporting dashboards
- Discounts, promotions, vouchers
